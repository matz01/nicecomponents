## React Component for rendering text

A basic div that renders some text

### Component usage

```js
<Chip text="hello from Chip" />
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<Chip text="hello from Chip" />
```

### Scss vars used

```css
--chips-bg: #999;
--chips-bg-selected: purple;
--chips-text: white;
--chips-text-selected: white;
```
